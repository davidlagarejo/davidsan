# Security Policy

La seguridad y la privacidad son parte central de este proyecto.

## Qué reportar

Reporta de forma responsable cualquier hallazgo relacionado con:

- exposición de secretos o credenciales
- fuga de datos personales
- configuraciones inseguras por defecto
- prompts o flujos que puedan filtrar información sensible
- dependencias con riesgos relevantes

## Qué no publicar en issues públicos

No compartas públicamente:

- tokens
- claves
- `.env`
- datos personales reales
- pasos que expongan información privada de terceros

## Alcance actual

Este repositorio es una base temprana. Aun así, cualquier problema de privacidad o seguridad se considera importante y será revisado con prioridad.
