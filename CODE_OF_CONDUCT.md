# Code of Conduct

## Our standard

Queremos una comunidad respetuosa, útil y segura para colaborar.

Se espera que toda interacción en este proyecto sea:

- respetuosa
- constructiva
- honesta
- abierta a crítica técnica sin ataques personales

## Conducta esperada

- da feedback sobre ideas y cambios, no sobre la persona
- asume buena fe y pide contexto antes de juzgar
- cuida el lenguaje cuando señales errores o riesgos
- protege la privacidad de otras personas

## Conducta no aceptable

- acoso o ataques personales
- publicación de datos privados o identificables
- lenguaje discriminatorio o intimidatorio
- spam, troleo o sabotaje deliberado

## Moderación

Los mantenedores pueden editar, ocultar o rechazar contribuciones e interacciones que dañen la seguridad, privacidad o salud de la comunidad.

## Alcance

Aplica a issues, pull requests, discusiones y cualquier espacio asociado con este proyecto.
